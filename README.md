# Practical IT Operations Tools

ICT System Engineer · PowerShell · Windows/Linux · Networks · Practical automation

I use GitHub for practical IT operations tools: administration scripts, network helpers, cleanup flows, and small command-line utilities that make recurring work easier to run, explain, and repeat.

## What You Will Find Here

- PowerShell utilities for system administration and support workflows.
- Network configuration and troubleshooting helpers.
- Python CLI tools for searching, auditing, and automation-adjacent tasks.
- Documentation-heavy projects with privacy, support, and handoff notes.

## Current Focus

- Making administration scripts easier to review, run, and maintain.
- Improving documentation and help text in my public repositories.
- Keeping automation careful: predictable defaults, clear prompts, and readable failure messages.
- Keeping private/internal infrastructure tooling documented without exposing details that do not belong in public repos.
